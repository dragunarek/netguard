SZYBKIE KOMENDY
INSTALACJA LOKALNEGO AGENTA PYTHON
Pełny agent Python działa wyłącznie lokalnie w Twojej sieci. Żadne dane nie opuszczają routera/komputera. Pobrany skrypt netguard_agent.py zawiera wszystkie moduły monitorowania.
WYMAGANIA SYSTEMOWE
✓ Python 3.9+
✓ Linux / macOS / Windows (WSL2)
✓ Dostęp do sieci lokalnej (LAN)
✓ Uprawnienia root/administrator
○ Opcjonalnie: Ollama + Llama3 (lokalny LLM)
○ Opcjonalnie: Home Assistant
ARCHITEKTURA AGENTA
netguard_agent.py
├── NetworkScanner — skanowanie ARP
├── PacketAnalyzer — analiza ruchu
├── ThreatDetector — wykrywanie zagrożeń
├── AIAnalyst — lokalny LLM (Ollama)
├── AlertManager — powiadomienia
└── WebDashboard — ten interfejs
SZYBKI START
# 1. Instalacja zależności
pip install scapy nmap-python psutil requests flask ollama
sudo python3 netguard_agent.py --setup
# 2. Uruchomienie agenta z lokalnym LLM
ollama pull llama3.2
sudo python3 netguard_agent.py --model llama3.2 --dashboard
# 3. Lub tryb bez LLM (reguły heurystyczne)
sudo python3 netguard_agent.py --no-llm --alert-email [email protected]
